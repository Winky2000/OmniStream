Custom header blocks

Use this to create optional blocks that appear under the newsletter logo/header. Each block is a single line (Text, Email, or URL). You can add multiple rows under the same header, and each row can show 1, 2, or 3 blocks. If all blocks are empty, OmniStream renders it as a header-only note (no blocks). You can still place {{CUSTOM_SECTIONS}} manually in a template, but if you omit it, OmniStream will auto-insert these blocks near the top.
